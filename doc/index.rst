Welcome to pyelectro's documentation!
=====================================
Tool for analysis of electrophysiology in Python.

This package was originally developed by Mike Vella. This has been updated by 
Padraig Gleeson and others (and moved to NeuralEnsemble) to continue development 
of pyelectro and Neurotune for use in OpenWorm, Open Source Brain and other projects
